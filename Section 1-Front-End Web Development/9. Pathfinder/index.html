Pathfinder

  1. Zero experience

    2. I know some of the later modules are really cool & there's a temptation to jump straight to it, but without the proper foundations, it's easy to get lost & demotivated.

    Make sure that you are completely comfortable with the concepts in each module of the curriculum before moving on.

    Usually, for beginners, I recommend spending a week on each module.

  2. Experienced Developer, New to Web

    3. You're probably here to quickly get up to speed with web development.

    Most people in this camp will have some experience with HTML & CSS, so I recommend starting with the JavaScript modules.

    As Javascript seems to live in its own land and has a number of weird features(/bugs), it might be a little shocking for people coming from safer languages (e.g. Swift).

  3. Experienced Web Developer

    4. Ok, you know Javascript, HTML and CSS like the back of your hand, but every week you hear about a new JS framework and you've decided it's finally time to buckle down and figure out what these kids are talking about.

    We'll start with Node and move on to Express, EJS, Postgres and more to get you quickly up to speed (and hopefully ditch PHP).
