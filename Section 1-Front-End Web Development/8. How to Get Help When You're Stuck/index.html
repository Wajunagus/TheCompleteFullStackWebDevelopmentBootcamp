How to Get Help When You're Stuck

With programming, it's almost a core part of the learning process that you do get stuck & you get yourself unstuck.

  1. Evaluate what is the expectation versus what is the reality.

    What did you expect your code to do & what actually happened & try to figure out what is causing the difference

  2. Search for any errors that you come across into Google & then heading towards the Stack Overflow websites that come up as a result.

    This is exactly what real life developers do every single day.

    It's all a part of the developer workflow.

  3. If that still doesn't help, or you're confused by the results that you're getting back, then it's a good idea to re-watch some of the video up to the point when the code started breaking or started doing something unexpected.

    Coz in the videos, there are always step by step instructions.

  4. You can check your code against my code.

  5. If that still doesn't solve it, then the next step is head towards the Q&A.

    6. 4 step approach to get the most helpful responses & solutions in the Q&A.

    1. What did you expect to happen

    2. What actually happened

    3. Screenshot of all your code that corresponds to the situation

    4. Screenshot of the debug console that includes any errors, if there were any errors, or if your app has crashed.

It's important to remember that everybody gets stuck sometimes. And just remember that getting stuck & solving it is all a part of the learning journey.
