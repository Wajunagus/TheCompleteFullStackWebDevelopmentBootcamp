The Box Model

Margin, Padding & Border

These 3 properties Margin, Padding & Border along with Width & Height, form a really important CSS concept known as the "Box model".

Well, if you think of a typical website here, like I Love Bacon, then you've got different HTML elements that are on screen.

And even though we can't really see it, effectively each element is a box in itself, & we can change the dimensions of those boxes by changing things such as width & height. And that invisible box around the elements describes how it would be laid out on screen.

We can change the height of an element to allow it to take up more vertical spacing on the web page. Similarly, we can also change the width of an element to allow it to take up more space horizontally, & we can either do this using pixels or percentages.

Border

In addition to width & height, we can also set a border on each of the HTML elements. And the way that the border property works is a little bit different from what you might have seen before. It can take 3 values separated by a space.

  1. The first value that you should provide is the thickness of the border.
  2. The second item is the style of the border. You can have for example, solid, but you can also have a dashed line.
  3. And finally, you've got the color of the border in which you can set to any named color or hex color as we've covered in the color properties lesson.

Remember, the important thing is that the height & the width of your HTML element, your box does not change. So the border actually goes outwards rather than inwards.

Now, once you've defined a border, you can further modify it with more specific rules.

This border width works Similarly to the way that the border property works, namely that it can take a maximum of 4 values.

  1. Top
  2. Right
  3. Bottom
  4. left

It goes in a clockwise direction.

In addition, you can also just provide two values for the border width.

Padding

Now, another property I want to talk about in regards to the box model is padding. And this is most easily seen when we have content such as text.

Margin

In addition to the padding, there's also a margin. And the margin is actually the part that's outside of the border. And it adds spacing between elements that you would not get with the padding.

Box Model

This box model is going to come really, really handy when you start to try & lay out different items on your website determining how they look & where they're placed relative to each other.

The final thing I wanted to mention is that padding & margin, similar to what we saw for the border & also the border-width, function in that same way where we've got 4 values that we can provide which will determine how they look on all 4 sides.

Again, with the border-width, you can change to only provide 2 values.

So this is just a particular way of how these 3 properties can accept multiple values.

Content Division Element

In addition to the boxes that are created by elements, we can actually create our own boxes independent of that, & that's really useful when we want to group different bits of content together so that we can style it together or maybe we want to lay it out together.

There was no point in learning about the div element before we could learn about CSS coz these are completely invisible unless you apply CSS style to it & the whole purpose is to act as an invisible box that can contain content in between the opening & the closing tags. In the middle, we can put as many elements as we want & we can group them into separate boxes known as "divs".
