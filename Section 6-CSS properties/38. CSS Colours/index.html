CSS Colors

Adding color to our websites

In this section we're going to do a deep dive into some of the most important CSS properties that you'll see & use. And we're going to start with CSS color properties.

Named Colors

There's a whole bunch of them out there with poetic names like cornflowerblue, cadetblue, dimgrey, olivedrab etc.

And you can find all of them by, of course, going to the Developer Documentation.

📜〈named-color〉

So always check the MDN Docs if you're not sure which property does what.

So you can simply copy the name of the color & paste it into you code & you'll be able to see that rendered in your website.

hex codes

Even though there's many colors available in the default CSS named colors, but you still sometimes might want to have a more unique color palette. So what do you do in that case? Well, one of my favourite color palettes is Color Hunt. It's a free tool that's created by designers for designers & of course, for us as well.

📜Color Hunt

Here you've got professionally designed color palettes for colors stacked together that would look perfect on any website.

But how do we use them if we don't have a name that's attached to it? Instead, we've got all these numbers & letters, what exactly are these?

Well, these are "hex codes" & they are a way of representing a larger number code for that particular color.

So, for example, in this case, the color purple which is formed of 3 fractions, red, green & blue. And these are colors out of 255.

So, a fraction, 93 out of 255 red, 56 out of 255 green & 145 out of 255 blue.

RGB Colour Sliders:

📜CSFG - RGB Colour Mixer - Used by Screens

And when you mix all 3 of these primary colors, you can get so many different types of shades.

So in this case, we've got 93, 56, & 145, & we end up with exactly the same shade as the purple.

Converting that RGB into a hex code, so a different way of representing that number, we get this particular hex code.
