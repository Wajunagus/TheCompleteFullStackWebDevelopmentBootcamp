Motivational Poster Project

Make your own memes

So for the final project, we're going to be building a motivational poster website & we're going to make our own memes.

You might have seen some of these around the Internet.

The important part of a motivational poster is;

  • there is an image that is going to be big

  • there is a border around the image

  • a black background

  • a big main title, which we're going to use a custom font to mimic this font a little bit better

  • and finally, we have a paragraph @ the bottom

💽
