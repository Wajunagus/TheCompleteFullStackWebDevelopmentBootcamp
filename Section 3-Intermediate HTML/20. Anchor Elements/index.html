The Anchor Element

Understanding HTML Attributes

In this lesson, we're going to understand a bit more about another aspect of HTML, which is their attributes, & to understand that we're going to learn all about the anchor element, which allows us to create hyperlinks.

The attribute for an HTML element goes in the opening tag. It goes into the part just after the name of the tag & just before the end of the opening tag where we have the closing angle bracket.

And what an attribute does is it adds additional information to this HTML element, such as in this case where this link should link to.

The way we structure an attribute

So we have the name of the element, & then we have a space, & then we have the name of the attribute. And afterwards, we have an = sign & after we have the value of the attribute.

Now, you can have as many attributes as you want & all you have to do is simply separate each one of them with a space & make sure that they all go into the opening tag.

In addition to the specific attributes which are relevant to a particular element, there are also global attributes that every single HTML element has access to. An example is "draggable". You can apply a global attribute to any given element, so in this case we can apply it even to our anchor element.

So we've learned that there are specific attributes which are reserved for only certain elements such as the "href" for the anchor tag, & there are global attributes which can be used on any HTML element.
