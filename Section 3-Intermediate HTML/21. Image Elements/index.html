The Image Element

Adding images to our websites

You'll notice that similar to the anchor tag or the anchor element that we learned about in the last lesson, the main component of an image tag that renders the image is the attribute source-"src". And what this does is it tells the image element what is the source of the image, & that's why after the = sign, the value is going to be the location of the image.

One last thing to notice about the image tag is, unlike the anchor tag, it doesn't have a closing tag, instead, it is a self-closing tag, also known as a void element. Coz if you think about it, what is the text content of an image?, doesn't really make sense, right? The main content is actually the image that will be rendered from the source.

It gives you a random image from its data bank.

The other attribute for an image tag that's super important & one that you should always have is the "alt" attribute. And the "alt" attribute stands for Alternative Text Description. This is really important coz people who are blind or visually impaired, they don't look @ the internet the same way we do. They often use a screen reader, & what it does when it hits as image is it will look @ the alternative text provided.

As an example, I'm using the Silktide Toolbar ⚓ https://silktide.com/toolbar/ which is a Chrome browser extension, to simulate what a person with visual impairement might hear as they're browsing.
