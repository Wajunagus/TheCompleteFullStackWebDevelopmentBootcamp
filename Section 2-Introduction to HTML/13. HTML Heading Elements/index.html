The Heading Element

‹h1› to ‹h6›

What is the purpose of heading elements?.

Well, the idea comes from book binding.

If you create a book & you create a table of contents, you'll see that there's a sort of hierarchy.

These levels are what we get to define using our HTML headings.

By default the different heading levels are styled to have different sizes. h1 being the biggest & h6 being the smallest. We can of course further style this later on.

‹Tag› vs Element

So what exactly is the difference?; anything that's inside an angle bracket is a tag while the element includes the content as well as any opening or closing tags.

✅ & ❎

Do's & dont's

One important rule; its not good practice to have more than one h1, coz the h1 is the very top-level heading.

Don't skip a level; don't go straight from h1 to h3. Don't jump between the different level headings.

They're rules for convention, & coz we're going to be professionals, we're going to start off on the right foot & learn all the rules that we need to know.
