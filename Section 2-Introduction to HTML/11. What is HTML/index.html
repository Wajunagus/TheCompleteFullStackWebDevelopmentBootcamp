What is HTML?

Introduction to HTML

No matter what browser you like to use, all of these applications do the same thing. They take different files such as HTML, CSS & JS, & render them into a website.

You can have just an HTML file, & in fact that's exactly what the first websites were created with.

HTML defines the content & structure of the website.

HyperText Markup Language: HyperText/hyperlinks; pieces of text which can link to other documents in the website. They are the foundation of how an HTML website works.

The world's first website created by Sir Tim Berners-Lee, who also happens to be the inventor of the internet. ⚓https://www.w3.org/History/19921103-hypertext/hypertext/WWW/TheProject.html: The website is filled with these hyperlinks marked in blue.
