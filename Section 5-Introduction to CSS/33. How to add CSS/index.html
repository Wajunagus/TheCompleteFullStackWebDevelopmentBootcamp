How to Add CSS

Inline, Internal & External CSS

There's 3 ways of adding CSS into an HTML website. Inline, Internal & External.

Inline

As the name suggests, it goes into the same line as a particular HTML element.

The inline CSS goes into the opening tag of the HTML. It's in the same line as the HTML tag.

You've got a style attribute, which is an attribute that is globally available to all tags. As we know, normally attributes have a name of the attribute, the = sign, and a value for the attribute. In this value-section is where we add our CSS code.

Our CSS code is broken down like this:

  1. The 1st part of it is the property that you want to change.

  2. And the 2nd part is the value of that property that you want to set it to.

Inline elements are really useful for adding CSS style to just a single element on your HTML page. But normally that's quite cumbersome coz as you can imagine, there's tons of different HTML elements. If you have more & more of them & you wanted them to share similar styles, then you would have to add these inline styles to each & every single one of them, which is pretty tedious.

It's not normally recommended to use inline styles in your entire document. It's only for specific sections, or when you're testing, or when you only want it in 1 single element, or 1 line in your HTML document.

Internal

This is done through a special HTML tag called the "style element." We've got an open  ‹style›  tag & a closing  ‹style/›  tag, & in between those 2 lines is where we add all of our CSS.

Our internal style can apply to anywhere within the same HTML document.

What we have to add in addition is what we call a "Selector," & this selector comes before a set of curly braces & then the CSS goes in between these set of curly braces.

Our Internal styling can apply to anywhere in our HTML document, & inside we can target or select as many elements as we want.

Internal styles are really useful for applying it only to 1 HTML document. The style & any code that goes in there is limited to the HTML that it sits in.

External

The biggest difference between the external, the internal, & the inline styles is that this actually lives in a completely separate file, which you'll normally see as styles.css or main.css or anything that has a .css extension. Inside this CSS file is where we write our CSS rules.

So we've got our selector, we've got our property & we've got values & we can make many, many of these inside this file.

Now how do we link up the style sheet file with our index.html? Well inside the head section of our HTML, we would add a link element which you'll notice is a self-closing tag, so it doesn't need a closing tag. And we have 2 things that we normally write.

  1. A relationship (rel) which refers to what is the role of this thing that we're linking to.

  2. The href, which is where is it located?

This style of external CSS is what's used most commonly in web development. And this is what we're going to be focusing on most of the time.

So remember, inline is really useful when you only want to target a single element, & internal is really useful when you only want to target a single web page. But for most cases & when you have a multi-page website, then what you're going to need is the external way of adding CSS through the use of a separate file, which will be called something, something.css. And it's inside this file where you're going to be writing the CSS code that can target an entire website with all of its multiple web pages.
