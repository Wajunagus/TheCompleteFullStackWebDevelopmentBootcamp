CSS Selectors

Choosing Where to Apply CSS

In this lesson, we're going to be talking about CSS selectors, & this is going to help us choose where to apply our CSS.

What is a CSS selector? Well, it's the part that selects the HTML in order to apply whichever rules go in between the curly braces.

Element Selector

This style of selector is one of the simplest CSS selectors, & all it does is it selects a particular HTML tag. So in this case it's targeting all h1s. If your HTML is any good, you should only have a single h1, but it will be different for other tags.

Class Selector

This requires a special symbol, which is the dot (.), & then after that dot immediately with no spaces, then you have the name of the class.

What is a class? A class is something that we can add as an attribute to any HTML element. And this attribute is kind of like saying, well, let's group these particular elements into a class. So it's used for grouping elements in your HTML file to apply the same CSS rule to all of them.

Now, what if we had multiple elements with the same class? Even though these 2 are completely different elements, we can tag them using the same class.

As you can see, the class selector is really versatile, it allows you to group different parts of your HTML file into having the same styling. And this can be really, really useful on multi-page websites with lots of different elements, & you don't just want to select a particular type of HTML element.

Id Selector

The ID Selector has its own special symbol, which is a pound sign or a hashtag (#), & you put that pound sign & again, no spaces in between the pound sign & the actual name of the ID, & this selects all elements with a particular ID attribute.

It works similarly to the class selector.

So then what is the difference between the ID & the class selector? Well, the class selector can be applied to many elements, whereas the ID should be only applied to 1 element in a single HTML file. In a single HTML file like our index.html, there should only be 1 ID of this particular name, "main". It should be completely unique & this is the difference.

Again, remember, IDs are unique, only 1 element per file and classes you can put on as many elements as you like to group them together.

Attribute Selector

Another way that you can select parts of your HTML is using the attribute selector.

So we know that we can create a HTML tag & add as many attributes as we like. Now, we can select the elements that have particular attributes or particular attributes values by using this notation. Notice the first part is the HTML element that you want to select, & then using a set of square brackets [], then inside we can include the attribute that you want to select.

Now, you can go 1 step further with attribute selectors. You can actually select the value of the attribute that you want to apply your CSS to.

This is again a really versatile way of applying CSS, but as you can see, it selects on different things than what we've seen before.

Universal Selector

It's really simple. It's just an asterix (*). And after the asterix you have your CSS rules, but what this means is select all.

And when you apply this, it doesn't matter what class you've got, what ID, what attribute set, which different elements, if you select all, it's going to apply the style to everything where the stylesheet is active. And this is probably one of the simplest selectors to understand.
