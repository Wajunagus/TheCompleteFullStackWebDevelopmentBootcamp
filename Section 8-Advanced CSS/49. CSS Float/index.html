And this brings me on to an important point about using float as we're doing here for layout.

In the past, a lot of developers have used float extensively & sort of wrangled it to create these kind of complex layouts.

Now I'm showing you this layout so you can use the clear property, but in modern web design & development, we don't have to use the float property to achieve these kind of designs coz there are much better tools for us like Flexbox, Grid, Bootstrap, a whole load of other things that is a lot less complicated to use & to understand than using float.

There can be very unexpected results if you're trying to achieve complex layouts using float.

So I recommend to only use float when you want to wrap text around an image, & in modern web design, this is what you're going to see most people doing as well.

So it's an important concept to learn & understand, but try to not use it on everything that you see. We've got better techniques, which I'll teach you very shortly.
